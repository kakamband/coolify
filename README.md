# Getting Started
New installation process soon!

## Updating process
### Update everything (proxy+coolify)
Updating proxy cause downtime!
-  `bash install.sh all`

### Update coolify only
-  `bash install.sh coolify`

### Update proxy only
Updating proxy cause downtime!
-  `bash install.sh proxy`
